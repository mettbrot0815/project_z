## Screenshots to Capture

When running the game locally, capture the following:

1. **Main Menu** - Title screen with game logo
2. **Gameplay 1** - Units moving on the battlefield
3. **Territory Capture** - Map showing owned territory
4. **Combat Action** - Units attacking enemies
5. **Victory Screen** - Win animation and summary
6. **Factory Menu** - Building interface

## Screenshot Quality Tips

- Use high-resolution display (1920x1080 or higher)
- Capture at 60 FPS for smooth gameplay shots
- Ensure proper lighting for UI shots
- Use consistent camera angles for gameplay shots

## Notes

- Screenshots are stored in `screenshots/` directory
- Use Godot's built-in screenshot command: `F12`
- For web builds, use browser developer tools
- Consider compression for web distribution
