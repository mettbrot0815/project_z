## Local Export Instructions

### Prerequisites
- Godot Engine 4.6.1 or later
- Export templates installed (optional for basic exports)

### Export Commands

**Windows:**
```bash
godot --export-release "Windows Desktop" project.godot
```

**Linux:**
```bash
godot --export-release "Linux/X11" project.godot
```

**Web (HTML5):**
```bash
godot --export-release "Web" project.godot
```

### Export Output Location
Builds will be created in the `builds/` directory:
```
builds/
├── Project_Z-v1.0-Windows.zip
├── Project_Z-v1.0-Linux.zip
├── Project_Z-v1.0-Web.zip
└── README.md
```

## Troubleshooting

**Issue:** Export fails with "Export templates not found"
**Solution:** Install export templates via Godot's Project Settings

**Issue:** Game crashes on startup
**Solution:** Verify all assets are present in the `assets/` directory

**Issue:** Poor performance
**Solution:** Lower graphics settings in Project Settings
